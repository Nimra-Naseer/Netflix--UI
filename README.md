# 🎬 Movie & Series Streaming UI

🍿 A modern, Netflix-inspired interface for exploring movies and series in one place.

## 🌟 About the Project
This project is a stylish frontend UI where users can browse trending movies and series.  
It includes different genres with a focus on **romantic ❤️** and **Islamic 🌙** content as part of personal taste and selection.

## 💡 Features
- 🎥 Movie & Series showcase UI
- 🍿 Trending section layout
- ❤️ Romantic content section
- 🌙 Islamic-themed content section
- 🎨 Clean and modern Netflix-like design

## 🚀 Purpose
To practice frontend design skills and create a visually appealing streaming platform UI concept.

## 🛠️ Tech Stack
- HTML
- Tailwind CSS

## 📌 Note
This is a personal UI project for learning and design practice. Content categories are based on personal preference.
